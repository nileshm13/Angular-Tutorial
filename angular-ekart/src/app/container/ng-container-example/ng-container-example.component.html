Ng-container is used when user wants to use multiple structural directives on the same element. We can assign if else on an ng-container but the else condition must be in a ng-template not an ng-container. Passing a template reference to ng-container in an else condition of ngIf gives an error

When using ng-template, we call it using *ngTemplateOutlet inside div, we can replace div by ng-container
